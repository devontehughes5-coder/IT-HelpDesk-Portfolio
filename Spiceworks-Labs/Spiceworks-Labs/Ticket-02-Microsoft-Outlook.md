# Ticket 02 – Microsoft Outlook Won't Open

## Scenario

A user contacted the Help Desk after Microsoft Outlook failed to launch following a Windows login.

---

## Issue

The user reported that Outlook would briefly appear and then immediately close. Restarting the workstation did not resolve the issue.

---

## Priority

Medium

---

## Status

Resolved

---

## Troubleshooting Steps

- Verified the issue with the user.
- Confirmed Windows login was successful.
- Checked Task Manager for existing Outlook processes.
- Attempted to launch Outlook in Safe Mode.
- Verified Microsoft Office installation.
- Performed Outlook profile repair.
- Restarted the Outlook application.
- Confirmed mailbox access.

---

## Resolution

Microsoft Outlook profile was repaired successfully. User confirmed Outlook opened normally and email access was restored.
