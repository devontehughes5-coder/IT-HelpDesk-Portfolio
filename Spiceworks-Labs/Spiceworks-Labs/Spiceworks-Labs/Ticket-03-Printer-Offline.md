# Ticket 03 – Printer Offline

## Scenario

A user reported that a network printer displayed an "Offline" status and documents remained stuck in the print queue.

---

## Issue

The user was unable to print documents to the shared office printer. Print jobs remained pending and the printer appeared offline.

---

## Priority

Medium

---

## Status

Resolved

---

## Troubleshooting Steps

- Verified printer was powered on and connected to the network.
- Confirmed workstation network connectivity.
- Cleared the Windows print queue.
- Restarted the Print Spooler service.
- Removed and re-added the network printer.
- Printed a successful test page.
- Confirmed the user could print documents successfully.

---

## Resolution

Printer communication was restored after restarting the Print Spooler service and reconnecting the printer. The user successfully printed a test document.
