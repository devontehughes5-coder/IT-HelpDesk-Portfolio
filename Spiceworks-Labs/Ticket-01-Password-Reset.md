# Ticket 01 – Password Reset

## Scenario

A user contacted the Help Desk after forgetting their Windows password and was unable to sign in.

---

## Issue

The user reported multiple failed login attempts and requested a password reset.

---

## Priority

Medium

---

## Status

Resolved

---

## Troubleshooting Steps

- Verified user identity before making account changes.
- Confirmed the user account was active.
- Reset the user's password.
- Provided a temporary password.
- Instructed the user to change the password after signing in.
- Confirmed successful login.

---

## Resolution

Password reset completed successfully. User was able to sign in and confirmed access to the workstation.
